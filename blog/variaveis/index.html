Variáveis

31/03/2014

É um nome simbólico criado pelo programador que representa um endereço de memória cujo conteúdo (valor) pode ser alterado durante a execução do programa.

Tipos de Variáveis

String Qualquer caracter,

byte O tipo de dados é um byte inteiro assinado complemento 8-bit dois. Ele tem um valor mínimo de -128 e um valor máximo de 127 (inclusive), pode ser usado para armazenar até mesmo idade, a não ser que alguém que use seu sistema possa ter mais que 127 anos.

short O tipo de dados short é um inteiro assinado de 16-bit complemento de dois. Use um short para economizar memória em matrizes de grandes dimensões, em situações em que as economias de memória realmente importa.

int Por padrão, o tipo de dados int é um inteiro de 32-bit.

long O tipo de dados long é um int de 64-bit com dois complementam inteiro. Use este tipo de dados quando você precisa de uma faixa de valores mais ampla do que os fornecidos pela int.

float O tipo de dados float é um de precisão simples de 32 bits, use um float (em vez de double), se você precisa economizar memória em grandes conjuntos de números de ponto flutuante.

double O tipo de dados double é uma dupla precisão 64-bit, (para valores decimais, este tipo de dados é geralmente a opção padrão).

boolean O tipo de dados boolean tem apenas dois valores possíveis: verdadeiro e falso.

char O tipo de dados char é um único caracter

Observação:

Tipo Valor padrão
byte 0
short 0
int 0
long 0L
float 0.0f
double 0.0d
char ‘\u0000’
String null
